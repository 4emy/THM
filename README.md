# The Human Manual (THM in short)

A comprehensive, single-file reference on the neuroscience and psychology of human behavior. 19 chapters covering the full stack of what makes humans who they are.

## What This Is

A self-contained `index.html` file with zero dependencies, frameworks, or build tools. Open it in any browser. That's it.
